# What is this?
This project is a service that you can self-host to provide language services like live closed caption generation and translation. 

This project is made to provide improved accessibility and cross-language communication for Resonite.

Resonite is a free social VR sandbox platform, which allows for socialization and collaborative in-game building. You can get Resonite free on Steam: https://store.steampowered.com/app/2519830/Resonite/

# Can I use this for my own project outside of Resonite?
Yes please! 

Our goal is to not only improve accessibility and communication on Resonite, but to raise the bar of accessibility across the industrty, which is why we made this fully open source, self-hostable and not tied specifically to Resonite.

# Why self hosted?
While there is a number of online API services that provide speech-to-text abilities, they can get quite expensive, as you have to pay for every minute of transcribed audio. Meaning any solution using these would need to be pay-gated. 

We don't belive that accessibility services should be hidden behind a pay-wall, but we also cannot offer processing for free (we'd still have to pay for it ourselves) - self hosting solves this problem!

# Why the name Babbelite?
It's three things mushed into one:
- The word "babel" from the "Tower of Babel" story which is associated with languages
- The word "babble" which is a cute way to describe talking
- Suffix "-ite" to make it sound like a mineral/crystal, which is part of core theme (both visual & naming) of Resonite

It also sounded funny to me and Google cannot find anything with this, so it was unique too.
